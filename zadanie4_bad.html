3)gdzie jest wykonywany JS (klient lub serwer)

Oprócz witryn i aplikacji, programiści mogą również używać JavaScriptu do tworzenia prostych serwerów internetowych i rozwijania infrastruktury zaplecza przy użyciu biblioteki Node.js, który zapewnia środowisko zawierające niezbędne narzędzia wymagane do uruchomienia JS na serwerach. JavaScript służy również do tworzenia gier przeglądarkowych. To świetny sposób dla początkujących programistów na ćwiczenie swoich umiejętności. Posiada różne biblioteki i frameworki do tworzenia gier, takie jak PhysicsJS, Pixi.js. Możemy również użyć WebGL (biblioteki grafiki internetowej), która jest API JS do renderowania obrazów 2D i 3D w przeglądarkach.
